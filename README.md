# **3D Modeling & Animation**
### Donut Project Overview

This project was built from scratch using Blender, showcasing a complete 3D production pipeline. The scene features detailed, realistic donuts layered with dynamic, flowing frosting.<br>
To bring the composition to life, I set up a fully animated camera path to capture the scene from multiple dynamic angles, paired with custom environment design and advanced lighting to create a rich, cinematic atmosphere.<br>
Following the modeling and animation phases, the entire sequence was fully rendered. 
The repository includes a short, high-quality [video clip](https://github.com/Nikolla04/Donut-blender-project/blob/main/Video%20snimak/video0001-0185.mp4)<br> showcasing the final rendered animation, along with the source `.blend` assets.
